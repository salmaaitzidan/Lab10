# Contribuer à NavigationDrawerDemo

## Comment contribuer

1. Fork le projet
2. Créer une branche (`git checkout -b feature/ma-fonctionnalite`)
3. Committer les changements (`git commit -m 'Ajout de ma fonctionnalité'`)
4. Pusher la branche (`git push origin feature/ma-fonctionnalite`)
5. Ouvrir une Pull Request

## Standards de code

- Respecter les conventions Java Android
- Commenter le code en français
- Tester sur API 24 minimum
